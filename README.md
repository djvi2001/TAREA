FECHA:01/03/2026

EL JUEGO DEL AHORCADO.

El motivo principal por el cual escogí este juego es por que me parecio interesante el desafío de lograr que el programa se construyera desde cero , tratando de resolver sus problematicas una a una a lo largo de este proyecto. Programar el ahorcado para mi no solo fue escribir un juego, sino entrenar la mente para pensar como programador y aplicar la teoría en un proyecto tangible.

Dentro de las principales funcionalidades que podran encontrar en mi codigo está la puesta en pantalla de un saludo inicial y lopción elegible de iniciar o no el juego. 
Tambien se podrá encontrar la selección de una palabra aleatoria con la cual se desarrollará el juego, la elección de letras por parte del usuario para competar la palabra aleatoria, y porsupuesto, para hacerlo mas desafiante, un contador de intentos , el cual, si llega a cero, te mostrará un mensaje de "GAME OVER".
